Pre Test

Q1) Unipolar RZ is a type of ______ encoding




Show Answer

Line



Q2) DC component is present in the encoded signal and there is loss of synchronization for long sequences of 0’s and 1’s in unipolar RZ



Show Answer

True



Q3) NRZ-L is a type of _____ line encoding.




Show Answer

Polar



Q4) NRZ codes are easy to engineer and it makes efficient use of bandwidth.



Show Answer

True



Q5) In NRZ-I
Statement 1: For each 1 in the bit sequence, the signal level is inverted.
Statement 2: A transition from one voltage level to the other represents a 1.




Show Answer

Both are true



Q6) Which of the following is not a Polar line encoding scheme?




Show Answer

AMI



Q7) Manchester encoding is a combination of which two line encoding schemes.





Show Answer

NRZ-L and RZ



Q8) How many levels does manchester line encoding involve?






Show Answer

Two levels with inversion in the middle of each bit



Q9) The polarities in NRZ format use.





Show Answer

Complete pulse duration



Q10) Which of the following combines the idea of RZ and NRZ-I?






Show Answer

Differential manchester



Q11) In which line encoding scheme 0 bit is represented by Zero level and binary 1 is represented by alternating positive and negative voltages.





Show Answer

AMI



Q12) AMI is a __________ type of line encoding scheme.






Show Answer

Bipolar



Q13) In Pseudoternary line encoding
Statement 1: 0 bit is represented by Zero level
Statement 2: Binary 1 is represented by alternating positive and negative voltages.
Which of the above are true.





Show Answer

Both are false



Q14) The ________ represents the number of bits sent per second, whereas the ______ defines the number of signal elements per second in the signal.






Show Answer

bit rate, baud rate
